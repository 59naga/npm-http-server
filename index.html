npmcdn is an experiment in serving files directly out of npm packages over a CDN. Try a URL in the format /package@version/path/to/file. A few examples:

  • http://npmcdn.com/expect@1.12.2/umd/expect.min.js
  • http://npmcdn.com/history@1.12.5/umd/History.min.js
  • http://npmcdn.com/react-router@1.0.0-rc4/umd/ReactRouter.min.js

https URLs also work, so you can use protocol-relative URLs if you want.
